First time? You need a free Google Cloud OAuth Client ID (one-time, ~10 min). Tap Setup steps below for the walkthrough.

One-time Google setup:
1. Go to console.cloud.google.com → create a project (any name).
2. APIs & Services → Library → search Google Drive API → Enable.
3. APIs & Services → OAuth consent screen → External → fill app name + your email → add yourself under Test users.
4. Credentials → Create Credentials → OAuth client ID → type Web application.
5. Under Authorized JavaScript origins add: https://mcdermottj639.github.io
6. Create → copy the Client ID → paste it above → Save & Connect.
Google will warn "unverified app" — that's expected for a personal tool; choose your account and Continue.
